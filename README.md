# 動作確認環境  
Ubuntu 22.04.2 LTS  
node v16.13.1  
npm 8.1.2

# Todoアプリ起動方法

## Setup

必要なライブラリのインストール:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## デバッグモードでの起動

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

コマンド入力後ターミナルに表示された指示に従ってアクセスしてください


## リリースモードでの起動

アプリケーションをビルド:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

ビルドされたアプリケーションをプレビュー:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```
